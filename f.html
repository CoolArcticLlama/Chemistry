Chemistry Guide 101

"#" - Examples

Definitions:

Chemistry - Study of composition & structure of matter and changef that occur in matter.

Combustion Reactions - Chemical reaction between substances, usually including Oxygen and usually accomplished by the generation of heat and light in the form of flame.

Synthesis Reactions - Reaction that combines 2 elements to make a new molecule or compound.

Decomposition Reactions - Basically the opposite of Synthesis, you break the molecule or compound down.

Single Replacement Reactions - Reaction that replaces one atom in the compound with a different one without taking the compound apart and putting it back together.

# Wood Burns -> Smoke, Heat

# Iron Rusting (Fe(Iron) + O2(Oxygen Gas)) -> FeO2(Iron Oxide)

# Water (H2(Hydrogen Gas) + O(Oxygen Molecule))

If a molecule has a 2 like O2 then it is a gas. Not everything that has a ratio of 2 is a gas like U2(Diuranium). Any element that has a ratio of 2 has a prefix of "Di" (Diiron = Fe2 = iron gas).

Types of Reactions

Combustion Reactions:

# C + O2 -> CO2(Carbon Dioxide)

# C3H8(Propaine) + 5 O2 -> 3 CO2 + 4 H2O

# 2 CH3OH(Methanol) + 3 O2 -> 2 CO2 + 4H2O

# C3H6O(Acetone) + 4 O2 -> 3 CO2 + 3H2O

ALL Combustion Reactions have a product of CO2 and H2O

Synthesis Reactions:

# 3 H2 + N2 -> 2 NH3(Ammonia)

# MgO(Magnesium Oxide) + CO2 -> MgCO3(Magnesium Carbonate)

#2 CO(Carbon Monoxide) + O2 -> 2 CO2

# 2 Na + Cl2 -> 2 NaCl(Sodium Chloride)

Decomposition Reactions

# 2 Fe(OH)3 (Ferric Hydroxide) -> Fe2O3(Ferric Oxide) + 3 H2O

# 2 H2O -> 2 H2 + O2

# 2 AgBr(Silver Bromide) -> 2 Ag + Br2

# H2CO3(Carbonic Acid) -> CO2 + H2O

Single Replacement Reaction

# 2 K + 2 H2O -> 2 KOH(Potassium hydroxide) + H2

# Mg + Cu(NO3)2(Copper Nitrate) -> Mg(NO3)(Magnesium Nitrate) + Cu

# Zn + 2HCl -> ZnCl2(Zinc Chloride) + H2

Cl2 + Cal2(Calcium Iodine) -> CaCl2 + I2
